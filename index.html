Range Properties

  • startContainer, startOffset => node and offset of the start
  • endContainer, endOffset => node and offset of the end
  • collapsed => true if the range starts and end at the same point (no content inside) otherwise false
  • commonAncestorContainer => parent element of all the nodes in range

Range Selection Methods

set range start

  • setStart(node, offset) => set start at position offset in node
  • setStartBefore(node) => set start before node
  • setStartAfter(node) => set start after node

set range end

  • setEnd(node, offset) => set end at position offset in node
  • setEndBefore(node) => set end before node
  • setEndAfter(node) => set end after node

In all these methods, node can be both a text or element node: for text nodes offset skips that many of characters, while for element nodes that many child nodes.

Methods to create range

  • selectNode(node) => set range to select the whole node
  • selectNodeContents(node) => set range to select the whole contents of node
  • collapse(toStart) if toStart=true set end=start
  • cloneRange()

Range editing methods

  • deleteContents
  • extractContents
  • cloneContents
  • insertNode
  • surroundContents
